Inline CSS

This is a simple HTML webpage created using inline CSS. It includes headings, paragraphs, and an image.

Types of CSS:-

There are three types of CSS-

  1. Internal CSS
  2. External CSS
  3. Internal CSS

Inline CSS:-

Inline CSS is used with the elements.

Internal CSS:-

Internal CSS is used just before the body section with a style tag and under the style tag elements are styled.

External CSS:-

External CSS are written as a separated file with named style.css and are inked in head section of the html document.
